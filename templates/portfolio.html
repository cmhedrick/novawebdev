The Dubro site was one of the first projects carried out by NOVA Web.

Our Dictionary was a project for the HILT Institute over at the Arlington Career Center. The educational web apploication allows for classes to host their own Dictionary. These dictionaries are built by students as they learn new jargon and vocabulary.

Our Storybook is another educational web application that is beloved by the english department in the HILT Institute at Arlington Career Center. The Web application allows classes to set up their own libraries for students to populate with their own works of literature. The web application allows for the linear development of stories as well as a for the development of contemporary of literature by giving users the ability to branch off from a page to persue a different plot line. Stories and branches are able to be viewed as an actual tree even to allow for easy navigation.
